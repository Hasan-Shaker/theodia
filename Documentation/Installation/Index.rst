Installation
============

This extension may be installed like any other extension, either from the TYPO3
Extension Repository (TER) or from Packagist.


.. _installation-composer:

Install extension via Composer
------------------------------

1. Go to your folder where the root :file:`composer.json` is located.
2. Type following command to get the latest version that runs on your TYPO3
   version:

   .. code-block:: bash

      composer require causal/theodia
